# Personal Library Management System

A C# console application to manage a personal collection of books.

## Features
- Add, update, delete, view books
- Search books by title or author
- Data stored persistently in SQLite
- Demonstrates OOP concepts: abstraction, inheritance, polymorphism, interfaces, composition

## Project Summary

The Personal Library Management System is a C# console application designed to manage a personal collection of books. The application demonstrates core Object-Oriented Programming principles, including abstraction, inheritance, polymorphism, interfaces, and composition.

Users can add, update, delete, view, and search books by title or author. Data is persisted using an SQLite database, allowing information to be stored and retrieved across application sessions. The project follows a multi-phase development approach and represents a complete, production-ready console application suitable for portfolio presentation.
